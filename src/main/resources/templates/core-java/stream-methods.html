Java Stream API Methods

The Stream API in Java provides a functional approach to processing collections of objects. Streams can transform, filter, and aggregate data using a pipeline of operations. Below is a list of common Stream methods along with examples.

1. filter()

The filter() method is used to exclude elements based on a given condition (predicate).

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5, 6);
List<Integer> evenNumbers = numbers.stream()
   .filter(n -> n % 2 == 0)
   .collect(Collectors.toList());

System.out.println(evenNumbers); // Output: [2, 4, 6]

2. map()

The map() method transforms each element of the stream into another form by applying a function.

List<String> words = Arrays.asList("java", "stream", "api");
List<String> uppercaseWords = words.stream()
   .map(String::toUpperCase)
   .collect(Collectors.toList());

System.out.println(uppercaseWords); // Output: [JAVA, STREAM, API]

3. forEach()

The forEach() method is used to iterate over the elements of the stream and perform an action on each element.

List<String> names = Arrays.asList("John", "Jane", "Tom");
names.stream()
   .forEach(System.out::println);

// Output:
// John
// Jane
// Tom

4. collect()

The collect() method is used to convert a stream into a different form, like a list, set, or another collection.

List<String> names = Arrays.asList("John", "Jane", "Tom");
Set<String> nameSet = names.stream()
   .collect(Collectors.toSet());

System.out.println(nameSet); // Output: [John, Jane, Tom]

5. reduce()

The reduce() method is used to perform a reduction on the elements of the stream, combining them into a single result (like sum or multiplication).

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);
int sum = numbers.stream()
   .reduce(0, Integer::sum);

System.out.println(sum); // Output: 15

6. sorted()

The sorted() method is used to sort the elements of the stream either naturally or using a comparator.

List<String> names = Arrays.asList("Tom", "John", "Jane");
List<String> sortedNames = names.stream()
   .sorted()
   .collect(Collectors.toList());

System.out.println(sortedNames); // Output: [Jane, John, Tom]

7. distinct()

The distinct() method is used to remove duplicate elements from the stream.

List<Integer> numbers = Arrays.asList(1, 2, 2, 3, 4, 4, 5);
List<Integer> distinctNumbers = numbers.stream()
   .distinct()
   .collect(Collectors.toList());

System.out.println(distinctNumbers); // Output: [1, 2, 3, 4, 5]

8. limit()

The limit() method is used to truncate the stream, keeping only the first N elements.

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);
List<Integer> limitedNumbers = numbers.stream()
   .limit(3)
   .collect(Collectors.toList());

System.out.println(limitedNumbers); // Output: [1, 2, 3]

9. skip()

The skip() method is used to skip the first N elements in the stream.

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);
List<Integer> skippedNumbers = numbers.stream()
   .skip(2)
   .collect(Collectors.toList());

System.out.println(skippedNumbers); // Output: [3, 4, 5]

10. findFirst()

The findFirst() method is used to return the first element of the stream (if present).

List<Integer> numbers = Arrays.asList(10, 20, 30, 40);
Optional<Integer> firstNumber = numbers.stream()
   .findFirst();

System.out.println(firstNumber.get()); // Output: 10

11. anyMatch()

The anyMatch() method checks if any element in the stream matches the given predicate.

List<String> words = Arrays.asList("apple", "banana", "orange");
boolean containsBanana = words.stream()
   .anyMatch(word -> word.equals("banana"));

System.out.println(containsBanana); // Output: true

12. noneMatch()

The noneMatch() method checks if none of the elements in the stream match the given predicate.

List<String> words = Arrays.asList("apple", "banana", "orange");
boolean noGrapes = words.stream()
   .noneMatch(word -> word.equals("grape"));

System.out.println(noGrapes); // Output: true

13. allMatch()

The allMatch() method checks if all elements in the stream match the given predicate.

List<Integer> numbers = Arrays.asList(2, 4, 6);
boolean allEven = numbers.stream()
   .allMatch(n -> n % 2 == 0);

System.out.println(allEven); // Output: true
